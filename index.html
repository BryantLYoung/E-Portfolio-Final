  • HedgeDocs

    Html, MarkDown, Material for MKDocs

    I have added detailed steps for importing models into Hedgehog Engine games to the HedgeDocs website through their GitHub repository. I contributed to this effort by creating and submitting documentation written in HTML and Markdown, along with supplementary materials for MCDocs. The steps outline the process of extracting and converting a character model, importing it into Blender, and rigging it to the original model's skeleton. I also provided resources for managing weights and preparing materials and texture. This documentation will serve as a valuable resource for modders and game developers who wish to import custom models into Hedgehog Engine games.
